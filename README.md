# 🚀 DSA Social API (Spring Boot)

A backend project that demonstrates **Data Structures & Algorithms using real-world REST APIs**.

---

## 🧠 Concepts Covered

| Feature | DSA Concept |
|--------|------------|
| Users API | HashMap (O(1)) |
| Posts API | ArrayList + Sorting |
| Comments | Tree (Recursion) |
| Undo Feature | Stack |
| Job Queue | Queue |
| Followers | Graph |

---

## ⚙️ Tech Stack

- Java 8
- Spring Boot
- REST APIs
- In-Memory Storage (No DB)

---

## ▶️ Run the Project

```bash
mvn spring-boot:run
```

---

## 🔥 API Endpoints

### Create User
POST /users

### Get User
GET /users/{id}

### Create Post
POST /posts

### Get Posts (Sorted)
GET /posts?sort=likes
